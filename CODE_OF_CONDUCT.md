# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful and constructive
- Welcome newcomers and diverse perspectives
- Focus on what is best for the community
- Avoid demeaning or insulting language

## Reporting

Report concerns by opening a private issue or contacting the maintainer via GitHub.

## Enforcement

Maintainers may remove comments, commits, code, issues, and other contributions that violate this standard.
